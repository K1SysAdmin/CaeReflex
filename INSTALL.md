# Install

CaeReflex requires Python >=3.10.

```bash
pip install -e .
pip install -e ".[server]"
pip install -e ".[mesh]"
pip install -e ".[vtk]"
pip install -e ".[gmsh]"
pip install -e ".[all,dev]"
```

Core install supports CLI, OpenFOAM text inspection, Gmsh `.geo` inspection, source-file records, exports, and CrossRef commands. Heavy scientific readers are optional.
