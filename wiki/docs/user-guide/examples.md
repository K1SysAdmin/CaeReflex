# Examples

Bundled examples are small, local, offline-first fixtures:

| Example | Purpose |
| --- | --- |
| `gmsh_minimal` | Core `.geo` parser example. |
| `openfoam_cavity_minimal` | OpenFOAM-like case-folder parser example. |
| `vtk_minimal` | Legacy ASCII VTK parser example. |
| `crossref_context` | Mock-first CrossRef metadata example. |
| `agent_workflow` | Agent consumption examples for context files and REST workflows. |

These examples demonstrate parser behavior only; they are not validation, convergence, mesh-quality, or design-safety evidence.
