# Safe use policy

CaeReflex output is structured evidence, not engineering validation.

Agents and users must not claim that CaeReflex:

- validates a simulation,
- proves convergence,
- assesses mesh adequacy,
- certifies an engineering result,
- establishes design safety,
- reads full papers from CrossRef metadata-only records.

Treat extracted facts as file-derived facts, inferred facts as tentative, and CrossRef records as literature context only.
