# openfoam_cavity_minimal

A minimal CaeReflex parser example using OpenFOAM-like case-folder structure.

This is not represented as a validated, converged, production, or solver-ready OpenFOAM simulation. It exists to test read-only parser behaviour.

Run:

```bash
caereflex inspect-openfoam examples/openfoam_cavity_minimal --out build/openfoam_case.json
```
